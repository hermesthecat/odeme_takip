Veritabanı Göç İşlemi

Talimatlar:

  1. Aşağıdaki "Verileri Dışa Aktar" düğmesine tıklayın.
  2. İndirilen JSON dosyasını "html_app/database" klasörüne "localstorage-export.json" adıyla kaydedin.
  3. Konsol üzerinden "php migrate.php" komutunu çalıştırın.
  4. Göç işleminin tamamlanmasını bekleyin.
